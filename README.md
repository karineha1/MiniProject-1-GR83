# MiniProject-1-GR83


## Getting Started

1. Download the **UCI Bike Sharing Dataset** zip file with the following link: https://archive.ics.uci.edu/dataset/275/bike+sharing+dataset
2. Extract the archive and locate the daily aggregated dataset `day.csv`.
3. Open the Jupyter notebook `Gr83_Mini_Project_1.ipynb` in **Google Colab**.
4. Upload `day.csv` to the notebook working directory under `/content`.
5. Run all cells.

## Requirements
This mini-project uses standard Python data science libraries: Numpy, Pandas, and Matplotlib.
